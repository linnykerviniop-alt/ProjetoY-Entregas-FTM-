# Projeto Y

## Descrição

Projeto Y é uma plataforma de entregas rápidas e seguras, desenvolvida para conectar clientes e entregadores em um sistema urbano eficiente. Oferece funcionalidades para solicitação de entregas, rastreamento em tempo real, histórico de pedidos e gerenciamento de perfil para usuários e parceiros.

## Funcionalidades

- **Autenticação**: Login e cadastro para clientes e parceiros (entregadores).
- **Solicitação de Entregas**: Interface para criar pedidos com endereços, tipo de serviço e cálculo de preço.
- **Rastreamento**: Mapa interativo com Leaflet para acompanhar entregas em tempo real.
- **Histórico**: Visualização de entregas passadas com filtros e avaliações.
- **Perfil**: Gerenciamento de informações pessoais e configurações.
- **Modo Parceiro**: Funcionalidades específicas para entregadores, como toggle de status online e alertas de solicitações.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação.
- **CSS3**: Estilização com variáveis CSS para tema dark.
- **JavaScript**: Lógica interativa e manipulação do DOM.
- **Leaflet**: Biblioteca para mapas interativos.

## Como Executar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web moderno.
3. A aplicação é totalmente client-side, não requer servidor.
